# AI LEARNING： One Small Step
**这是一个简单的技术科普教程项目, 主要聚焦于解释一些有趣的, 前沿的技术概念和原理. 每篇文章都力求在 5 分钟内阅读完成.**

> 本站部分至少来自开源，致敬开源作者，感谢分享知识！[源仓库](https://github.com/karminski/one-small-step) ⭐

## 快速导航

### 人工智能相关
探索大语言模型、Transformer、注意力机制、微调技术等 AI 相关概念

### 数学相关
理解矩阵、拟合等数学基础概念

### 系统相关
了解 Windows 系统工具和内存管理

### 硬件相关
深入 PCIe、NVMe、内存等硬件技术
